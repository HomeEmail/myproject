如何获取账号？

农加网目前不支持在线注册账号。如果您想要注册农加网账号，请携带相关身份证明至线下的 服务站点>>进行账户注 册。其中：

    个人：

  • 个人身份证；

    企业/组织：

  • 组织机构代码证（原件/电子扫描件）;
  • 营业执照（原件/电子扫描件）;
  • 法人身份证；

忘记密码怎么办？

农加网目前不支持在线重新设置密码。如果您忘记登录密码，请携带账户绑定的有效身份证原件前往线下的 服务站点>> 办理账户密码重置业务。其中：

  • 个人用户：

  • 带个人身份证；

  • 企业/组织用户：

  • 法人身份证

什么是土地流转？

土地流转，即农村土地经营权的流转交易，是指农村集体经济组织、农业生产经营组织、法人和自然人之间依法对农村土地经营权进行：转让、入股、出租、互换、转包、抵押及其他交易行为。

如何办理土地转出申请？

农加网采用和政府合作的O2O模式开展相关的土流业务办理。如果您想要办理土地转出申请，即农村土地经营权转出申请，请按照如下流程进行办理：

1. 准备资料

根据土地转出的业务要求准备好相关材料（见土地转出相关材料）；

2. 前往服务点

携带业务材料至附近的 服务站点>> 进行办理；

3. 登记办理

业务人员将会为您登记相关材料申请办理，并开通/关联您的农加网账号；

4. 查看进度

回到家中您可以登录农加网账号查看业务办理的情况及进度；

5. 申请通过，信息公告

申请通过后，您的土地转出信息将会公告在农加网、产权交易中心电子大屏、各村村委公告栏上；

6. 确认转入方资格

公告期间，若有人申请转入您的土地经营权，农加网联合农村产权交易中心在审核其基本资格通过后将会通知您进行确认：是否同意申请者具备转入您土地经营权的资格；

7. 进行交易

公告期结束后，农加网联合农村产权交易中心将会组织您和相关具备转入资格的人员进行现场交易并签订合同；

8. 领取证书

最后，由农村产权交易中心登记此次交易的合同并为您和转入方出具《农村土地经营权流转交易鉴证书》。

土地转出申请需要哪些材料？

办理土地转出申请要求的材料包括：

1. 申请人的身份证、土地承包经营权证或其他权属证明材料、土地承包合同的正本和复印件；

2. 转让公告。现场填写转让公告，公告中披露标的的基本情况、标的底价及作价依据交易条件、受让方资格条件、对土地承包经营权交易有重大影响的相关信息等内容；

3. 村集体经济组织申请流转交易的，提交本集体经济组织成员大会通过的决议书。成员大会应当由本组织内具有选举权的成员的半数以上参加，或者本组织三分之二的户的代表参加，所做决定应经到会人员的半数以上通过。

4. 再流转交易的转让方必须提供原流转交易合同和《农村经营权流转交易鉴证书》或其它有效证明，转让方为自然人的，须提供身份证明；转让方为法人或其他组织，须提供法定代表人身份证明、组织代码证及工商营业执照复印件；

5. 需要评估的必须提供评估报告文书；

6. 委托经办人、发包方或中介机构办理交易手续的，需提交委托书及受托方主体资格证明、法定代表人或负责人身份证复印件；　

7. 其他需要提交的材料。

如何办理土地转入申请？

农加网采用和政府合作的O2O模式开展相关的土流业务办理。如果您想要办理土地转入申请，即农村土地经营权转入申 请，请按照如下流程进行办理：

1. 准备资料

根据土地转入的业务要求准备好相关材料（见土地转入相关材料）；

2. 前往服务点

携带业务材料至附近的 服务站点>> 进行办理；

3. 登记办理

业务人员将会为您登记相关材料申请办理，并开通/关联您的农加网账号；

4. 查看进度

回到家中您可以登录农加网账号查看业务办理的情况及进度；

5. 申请通过，进行交易

申请通过后，农加网联合农村产权交易中心将会在转入土地相应的公告期结束后，组织您和土地的转出方进行现场交易并签订合同；

6. 领取证书

最后，由农村产权交易中心登记此次交易合同并为您和转出方出具 《农村土地经营权流转交易鉴证书》。

土地转入申请需要哪些材料？

办理土地转入申请要求的材料包括：

1. 身份证明，个人客户提供个人身份证，企业/组织客户提供法人身份证、组织机构代码证、营业执照；

2. 受让申请，客户到现场填写登记相关申请信息；

3. 受让方的资信证明，包括：财务状况证明、支付能力证明、信用状况证明、农业经营能力证明等；

4. 委托代理的，需提交经公证的授权委托书及受托人身份证；

5. 采取联合受让的，应提交联合受让各方签订的联合受让协议；

6. 其他需要提交的材料。

什么是土地流转交易鉴证书？

土地流转交易鉴证书，即《农村土地经营权流转交易鉴证书》，是指经由农村土地产权交易中心对农村土地流转交易过程进行认证并最终出具给交易双方的确定土地经营权的证书。土地流转交易鉴证书确立了流转土地的经营权，维护了流转交易双方的合法权益，是对农民财产性权益的保护，是农地抵押贷款的重要载体。

如何办理土地流转交易鉴证书？

在农加网上登记办理土地流转的相关客户：

在土地流转交易完成，即签订合同后，将会获得由农村产权交易中心出具的 《土地流转交易鉴证书》。

未在农加网上登记办理土地流转的相关客户：

需要土地流转双方携带 土地转出相关材料，土地转入相关材料及土地流转交易合同到附近的服务站点>>进行办理。

什么是土地经营权抵押？

农村土地经营权抵押贷款是指借款人在不改变土地所有权、承包权性质，不改变农村土地农业用途的条件下，将农村土地经营权及地上附着物作为抵押担保向银行等相关金融机构申请办理的借款业务。

如何办理土地经营权抵押？

农加网联合政府、银行采用O2O模式开展农村土地金融业务的办理。如果您想要办理土地经营权抵押，即农村土地经营权抵押贷款，请按照如下流程进行办理：

1. 准备资料

根据土地经营权抵押的业务要求准备好相关材料（见土地经营权抵押相关材料）；

2. 前往服务点

携带业务材料至附近的 服务站点>> 进行办理；

3. 登记办理

业务人员将会为您登记相关材料申请办理，并开通您的农加网账号；

4. 查看进度

回到家中您可以登录农加网账号查看业务办理的情况及进度；

5. 申请通过，银行受理

申请通过后，将由受理您贷款的银行联系您进行面签；

6. 抵押登记，领取证书

前往附近的 服务站点>> 进行 土地经营权抵押登记 并获取《土地经营权他项权利证书》；

7.凭证拿款

凭 《土地经营权他项权利证书》到银行获取贷款。

土地经营权抵押需要哪些材料？

以家庭承包方式取得农村土地经营权作抵押的需提供：

1. 农村土地承包合同及土地承包经营权证。

2. 农村土地承包经营权证载明的承包人同意抵押、处置的书面证明；有共有权人的，需提供共有权人同意抵押、处置的书 面证明。

通过缴付租金方式取得农村土地经营权作抵押的需提供：

1. 承包合同或流转合同等合法取得农村土地承包经营权的证明材料。

2. 农村土地承包经营权证或经鉴证的农村土地承包经营权流转合同。

3. 农村土地承包经营权租金支付票据或相关证明。

4. 农村土地承包经营权证载明的权利人同意抵押、处置的书面证明；有共有权人的，需提供共有权人同意抵押、处置的书 面证明。

5. 农村土地承包经营权直接或间接从承包人处流转取得的，需提供承包人农村土地承包合同，承包人及共有权人同意抵 押、处置的书面证明

6. 农村土地承包经营权直接从村集体或其他集体经济组织承包取得的，需提供村集体或其他集体经济组织成员的村民会 三分之二以上成员或者三分之二以上村民代表的同意抵押、处置的书面证明。

什么是土地经营权抵押登记？

农村土地经营权抵押登记，是指土地承包经营者在不改变土地所有权性质、不转移农村土地占有和农业用途的条件下，将依法取得的土地承包经营权作为抵押担保向银行业金融机构申请贷款而发生的登记行为。当借款人到期不履行债务时，抵押权人有依法处置该农村土地承包经营权并用处置所得价款优先受偿的权利。

如何办理土地经营权抵押登记？

土地经营权抵押登记分为：土地经营权抵押初始登记、土地经营权抵押变更登记、土地经营权注销登记。客户根据自身实际需求进行选择办理。 农加网联合政府、银行采用O2O模式开展农村土地金融业务的办理。如果您想要办理土地经营权抵押登记，请按照如下流程进行办理：

1. 准备资料

根据土地经营权抵押的业务要求准备好相关材料（见土地经营权抵押登记相关材料）；

2. 前往服务点

携带业务材料至附近的 服务站点>> 进行办理；

3. 登记办理

业务人员将会为您登记相关材料申请办理，并开通/关联您的农加网账号；

4. 查看进度

回到家中您可以登录农加网账号查看业务办理的情况及进度；

5. 申请通过，领取证书

申请通过后，将由农村产权交易中心出具《土地经营权他项权利证书》或其他证明文件。

土地经营权抵押登记需要哪些材料？

抵押初始登记需要提供：

1. 现场填写的《蕉岭县农村土地承包经营权抵押登记申请书》；

2. 申请人身份证明：个人提供个人身份证，企业提供法人身份证、组织机构代码证、营业执照；

3. 抵押权人与抵押人签订的《抵押合同》；

4. 《农村土地承包经营权证》或《农村土地承包经营权流转合同》或《农村土地承包经营权流转鉴定书》；

5. 抵押物共有权人同意抵押、处置的书面资料；

6. 抵押物所在发包方同意抵押、处置的证明材料。

抵押变更登记需要提供：

1. 现场填写的《蕉岭县农村土地承包经营权抵押登记申请书》；

2. 申请人身份证明：个人提供个人身份证，企业提供法人身份证、组织机构代码证、营业执照；

3. 发生变更的书面证明资料；

4. 抵押权人同意变更证明；

5. 《农村土地承包经营权他项权利证书》。

抵押注销登记需要提供：

1. 现场填写的《蕉岭县农村土地承包经营权抵押登记申请书》；

2. 申请人身份证明：个人提供个人身份证，企业提供法人身份证、组织机构代码证、营业执照；

3. 抵押权因主债权履行完毕、主债权消灭、抵押权已经实现、抵押权人放弃抵押权、提前解除合同或者法律、法规规定抵 押权消灭的其他情形注销的证明材料；

4. 《农村土地承包经营权他项权利证书》。
